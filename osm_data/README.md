# Transit Data

This data will be utilized by the Valhalla path planner. Querying this data can be done as such:
```bash
wget -O colorado.osm.pbf https://download.geofabrik.de/north-america/us/colorado-latest.osm.pbf
```

More street map data can be found on [Geofabric's website](https://download.geofabrik.de/).
